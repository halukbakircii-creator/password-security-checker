
# Password Security Checker 🔐

A Python tool that analyzes password strength and provides security recommendations.

## Features

- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers
- Detects special characters
- Provides feedback to improve password security

## How It Works

The program evaluates the password using several security criteria and assigns a strength level:

- Weak
- Medium
- Strong

It also gives suggestions on how to improve the password.

## Technologies Used

- Python
- Regular Expressions (re)

## Example

Input:

Password123

Output:

Password Strength: Medium

Suggestions:
- Add special characters
